
# Chat Application
This is the application for chatting in a room.

# Overview
* front - frontend
* api - backend

# Features
* Real-time Chat
* Multiple Users Available

# Requirement
* Node.js 10.2.4
* Next.js 14.1.4
* .NET Framework 3.5

# Usage

Go to api directory and execute "dotnet run", and front directory to execute "npm start".
Then, visit http://localhost:3000/.
